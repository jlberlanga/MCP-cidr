# tf-ip-tools
CIDR calculator - ipaddress &lt;-> cidr


You can use as MCP/Tool in any client, eg: LMStudio or 

** LMStudio **


"mcpServers": {
  "tf-ip-tools": {
    "command": "python",
    "args": [
      "server.py"
    ]
  }
}


** OPENCODE ** 
 
"mcp":{
  "tf-ip-tools": {
    "type": "local",
    "enabled": true,
    "command": [
      "python",
      "server.py"
    ]
  }
}
